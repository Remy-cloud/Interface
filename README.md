# Unity Interface Assignment

## What This Project Does
This project demonstrates the use of **interfaces** in C# within Unity.
An `IShape` interface is defined and implemented across three shape classes.

## shapes
- **Trapezium** - Calculates area, perimeter, and unknown slanted side
- **Circle** - Calculates area, perimeter, and radius
- **Nonagon** - Calculates area, perimeter, and number of sides

## How To Run
1. Open the project in Unity
2. Attach `InterfaceTestScript` to any GameObject
3. Press Play
4. Open the Console to see the output
